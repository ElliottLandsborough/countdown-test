# Countdown Test

## Run with docker
```
$ git clone git@gitlab.com:elliottlan/countdown-test.git
$ cd countdown-test
...
```

## Simple dev server
```
$ git clone git@gitlab.com:elliottlan/countdown-test.git
$ cd countdown-test
$ composer install
$ php bin/console server:start
```
